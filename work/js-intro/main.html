    Embedded C Language

    The main core of embedded C language is the understanding of the registers in the chip, the timing of various signals of peripherals, and the understanding of signal transmission. Rational deployment of registers for digital data, on-chip resources, on-board resources, used to achieve a stable hardware system.

    Verilog

    Verilog is a language commonly used for FPGA programming and can be used for parallel programming. It can be used to design digital chips, to design high-speed acquisition systems, for the design of various parallel systems, or for the design of complex timings. Another corresponding programming language is VHDL.

    Assembly

    Assembly is a relatively low-level language compared to most languages, and provides insight into assembly language by the number of lines of code running in memory. Assembly is more cumbersome than other languages in terms of code length, scheduling, and interrupts, but assembly provides insight into how the CPU operates.
